POST Create New User
https://evolent-exercise.herokuapp.com/api/users

This end point will be going to use to create new user in the DB.

On Sucess : Will send the message as "User Created Successfully".

On Error : 500 : with error message in json format

GET Get All Users
https://evolent-exercise.herokuapp.com/api/users

Will get all the available users in database.

On Success : Send the users details in Array. On Error : 500 : with error in json format.



Example Request
Get All Users 
curl --request GET \
--url https://evolent-exercise.herokuapp.com/api/users
GET Get Single User Details
http://evolent-exercise.herokuapp.com/api/users/5b6b201a896a0400210d9701

Will send the all the details of user base on the UserId pass in the parameters.

On Success : User details in json format. On Error : 500 - Error with message in json format. Will need to make more improvement in error handling.

PUT Update Single User
http://evolent-exercise.herokuapp.com/api/users/5b6b201a896a0400210d9701

On Success : Message with successfully created user On Error : 500 - Error with message in json format. Will need to make more improvement in error handling.
